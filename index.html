Теоретические вопросы

  1. Как создать дату 24 января 2021 года, 22 часа 51 минута? Временная зона — местная.

    Ответ: let date = new Date(2021, 0, 24, 22, 51);

  2. Для чего предназначен метод getDay()?

    Ответ: это метод позволяет получить день недели от 0 (воскресенье) до 6 (суббота) из объекта Date в соответствии с местным часовым поясом.

  3. Как посчитать, сколько секунд осталось до завтра?

    Ответ: const getSecondsToTomorrow = () => {
    let today = new Date(); //объявляем переменную с сегодняшней датой
    let hour = today.getHours(); //получаем количество часов прошедших в сегодняшнем дне
    let minutes = today.getMinutes(); //получаем количество минут прошедших в сегодняшнем дне
    let seconds = today.getSeconds(); //получаем количество секунд прошедших в сегодняшнем дне
    let totalSecondsToday = (hour * 60 + minutes) * 60 + seconds; //вычисляем общее количество секунд прошедших сегодня
    let totalSecondsInADay = 86400; //объявляем переменную с общим количеством секунд за день
    return totalSecondsInADay - totalSecondsToday; //получаем оставшееся количество секунд до завтра
    };
    console.log(getSecondsToTomorrow()); //выводим результат вычисления функции в консоль

  4. Для чего предназначен метод `getDate()`?

    Ответ: это метод позволяет получить день месяца, от 1 до 31 из объекта Date.

  5. Что выведет console.log(d) ? let d = new Date(2016, 2, 9);
    console.log(d);

    Ответ: выведет дату 9 марта 2016 в местном часовом поясе
    Date Wed Mar 09 2016 00:00:00 GMT+0300 (Москва, стандартное время)

  6. Что делает getTimezoneOffset() ?

    Ответ: этот метод возвращает разницу в минутах между местным часовым поясом и UTC (всемирное координированное время)

  7. Что выведет консоль? let date = new Date();
    console.log(date.getUTCHours());

    Ответ: 20 часов по временной зоне UTC+0

  8. Для чего предназначен метод getHours() объекта Date?

    Ответ: для получения количества часов по местному времени на настоящий момент времени.

  9. Что выведет консоль?
    let d = new Date();
    let y = d.getFullYear();
    console.log(y);

    Ответ: получим текущий год в формате четырех цифр (2023)

  10. В чём ошибка в коде?
    let y = new Date();
    let d = y.getDate();
    console.log(d);

    Ответ: да, вроде, нет ошибки. Получаем сегодняшний день месяца: 13
